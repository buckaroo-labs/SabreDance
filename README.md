# SabreDance
Setup and friendly GUI for [sabre/dav](https://github.com/sabre-io/dav)

Implements the instructions found at https://sabre.io/dav/install/ and https://sabre.io/dav/gettingstarted/ and includes corrections to example code. Requires MySQL (SQLite option supported by sabre may be included here later).

Tested on PHP 8.1
